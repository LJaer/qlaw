###微信小程序体验
学习了别人的源码，感谢分享：https://github.com/zarknight/qlaw

简单法律手册，使用微信小程序俱乐部(http://www.wxappclub.com/)的接口进行数据模拟

1、使用wx.request来请求数据

2、this.setData来保存数据

3、onLoad:function(params)，params是接收页面传过来的数据
